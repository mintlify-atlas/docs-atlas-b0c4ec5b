# Welcome to Skiff

Skiff is a privacy-first, end-to-end encrypted workspace that empowers free and effective communication. This comprehensive suite includes four main applications designed to protect your digital privacy while maintaining a seamless user experience.

![Skiff Mail Header](https://github.com/skiff-org/skiff-mail/assets/3527315/9b8b7ee9-9daa-4e4c-b6bc-9a869d574757)

## What is Skiff?

Skiff provides a complete workspace solution where **all your data is end-to-end encrypted**. Unlike traditional productivity tools, Skiff never stores unencrypted access to your messages, files, documents, or calendar events. You can sign up, use, and even upgrade your account completely privately - even using an anonymous crypto wallet.

## Core Privacy Principles

<Info>
All messages sent between Skiff Mail users are end-to-end encrypted. Additionally, Skiff never stores unencrypted access to messages sent or received from external mail providers.
</Info>

Skiff's privacy model is built on several key principles:

- **Zero-knowledge architecture**: Your encryption keys never leave your device unencrypted
- **End-to-end encryption**: All data is encrypted on your device before being sent to Skiff servers
- **Open source**: The codebase is publicly available for security auditing
- **Privacy-first upgrades**: Pay with cryptocurrency for complete anonymity

## Technology Stack

Skiff is built using modern web technologies and cryptographic libraries:

- **Frontend Framework**: React 17 with TypeScript
- **Rich Text Editor**: ProseMirror with TipTap extensions
- **Cryptography**: Custom `skiff-crypto` library based on TweetNaCl and ChaCha20Poly1305
- **State Management**: Redux Toolkit
- **GraphQL API**: Apollo Client
- **Search**: MiniSearch with client-side encrypted indexing

<Note>
Skiff's technical implementation is documented in the [Skiff whitepaper](https://skiff.com/whitepaper), which provides detailed information about the cryptographic protocols and security architecture.
</Note>
